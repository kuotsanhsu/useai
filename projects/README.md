# Projects

Use this directory for project-facing knowledge that helps restart, inspect, guide, or improve a concrete project.

A project area may include:

- `map.md`: the current shape of the project.
- `open-questions.md`: unresolved issues, uncertainties, and things to test.
- `decisions.md`: decisions made, why they were made, and what would change them.
- `improvement-log.md`: concrete improvement ideas and follow-up actions.

Keep project notes oriented toward later action, not exhaustive record keeping.
