# Concepts

Use this directory for ideas that travel across projects: distinctions, models, references, arguments, and reusable ways of seeing a problem.

Concept notes should make later retrieval easy. Prefer a clear title, a short explanation, where the idea came from, where it might apply, and what remains uncertain.

Relevant fields include philosophy, psychology, economics, psychoanalysis, computer science, systems engineering, and operations research.
